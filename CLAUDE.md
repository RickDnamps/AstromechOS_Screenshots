# AstromechOS_Screenshots

Repository dedicated to hosting screenshots for:
- [AstromechOS](https://github.com/RickDnamps/AstromechOS)
- [AstromechOS_Imager](https://github.com/RickDnamps/AstromechOS_Imager)

The goal is to keep the main repositories lightweight by offloading media files here.

## Repository layout

```
AstromechOS_Screenshots/
├── Screenshots_Astro/      # Screenshots for the AstromechOS project
└── Screenshots_Imager/     # Screenshots for the AstromechOS_Imager project
```
