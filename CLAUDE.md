# AstromechOS_Screenshots

Dépôt dédié à l'hébergement des captures d'écran (screenshots) liées aux projets :
- [AstromechOS](https://github.com/RickDnamps/AstromechOS)
- [AstromechOS_Imager](https://github.com/RickDnamps/AstromechOS_Imager)

Le but est d'alléger la structure des dépôts principaux en y déportant les médias.

## Structure du dépôt

```
AstromechOS_Screenshots/
├── Screenshots_Astro/      # Captures d'écran du projet AstromechOS
└── Screenshots_Imager/     # Captures d'écran du projet AstromechOS_Imager
```

## Conventions Git — IMPORTANT

**Ne jamais ajouter de trailer `Co-Authored-By: Claude` (ou tout autre co-auteur Claude/Anthropic) dans les messages de commit.**

Tous les commits doivent être attribués uniquement à l'utilisateur. Ne pas inclure non plus de mention "Generated with Claude Code" dans les messages de commit ou les descriptions de PR pour ce projet.
